Note Taker Web Application

The Note Taker is a Java-based web application that allows users to create, view, update, and delete notes. 
The project is built using JSP, Servlets, Hibernate ORM, and MySQL, following the MVC architecture pattern.

🚀 Features
Add new notes
View all saved notes
Update existing notes
Delete notes
Hibernate-based ORM mapping
Clean and responsive UI using Bootstrap

🛠️ Technologies Used
Java (Core Java)
JSP & Servlets
Hibernate (ORM Framework)
MySQL
Apache Tomcat Server
Bootstrap

🧱 Architecture
This project follows the MVC pattern:
Model → Note Entity + Hibernate
View → JSP Pages
Controller → Servlets

📂 Key Concepts Implemented
Entity Mapping using Hibernate Annotations
Session & Transaction Management
HQL Queries
CRUD Operations
MVC Architecture
